## Usage

`BarsList` component displays a list of bars with names and values.
The only required prop is `data` â€“ an array of objects with `name` and `value` properties.

<Demo data={BarsListDemos.usage} />

## Data format

`BarsList` expects data in the following format:

```tsx
export const data = [
  { name: 'Google', value: 1200000 },
  { name: 'Direct', value: 100000 },
  { name: 'Bing', value: 200000 },
  { name: 'Yahoo', value: 20000 },
];
```

## Value formatter

Use `valueFormatter` prop to format the value displayed next to the bar.
