## Usage

Use the Highlight component to highlight a substring in a given string with a mark tag.

Pass the main string as children to the Highlight component and the string part that should be highlighted to the `highlight` prop.
If the main string does not include the `highlight` part, it will be ignored.
`Highlight` ignores trailing whitespace and highlights all matched character sequences.
