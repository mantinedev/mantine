## Usage

Set the `sections` prop to an array of:

- `value` – number between 0 and 100 – amount of space filled by the segment
- `color` – segment color from the theme or any other CSS color value

<Demo data={RingProgressDemos.usage} />

## Size, thickness & rounded caps

Use the `size`, `thickness` & `roundCaps` props to configure the RingProgress size and thickness values:

<Demo data={RingProgressDemos.configurator} />

## Sections tooltips

Add a `tooltip` property to a section to display a floating [Tooltip](/core/tooltip/) when the user hovers over it:
