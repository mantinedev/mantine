## DatePicker props

`DateTimePicker` supports most of the [DatePicker](/dates/date-picker/) props.
Read through the [DatePicker](/dates/date-picker/) documentation to learn about all component features that are not listed on this page.

## Usage

<Demo data={DateTimePickerDemos.usage} />

## Range

Set `type="range"` to allow selecting a date and time range. In range mode,
two time inputs are rendered for start and end times, and a range summary is displayed below:

<Demo data={DateTimePickerDemos.range} />

## Presets

Use the `presets` prop to add custom date presets. Presets are displayed next to the calendar:

<Demo data={DateTimePickerDemos.presets} />

## TimePicker props

You can pass props down to the underlying [TimePicker](/dates/time-picker/) component
with the `timePickerProps` prop. Example of enabling dropdown and setting `12h` format
for the time picker:

<Demo data={DateTimePickerDemos.timePickerProps} />

## Value format

Use the `valueFormat` prop to change the [dayjs format](https://day.js.org/docs/en/display/format) of the value label:

<Demo data={DateTimePickerDemos.format} />

`valueFormat` also accepts a function that receives the value as a `YYYY-MM-DD HH:mm:ss` string
and returns a custom formatted value. Use it when the formatting logic cannot be expressed with a
dayjs format string:

<Demo data={DateTimePickerDemos.formatFunction} />
