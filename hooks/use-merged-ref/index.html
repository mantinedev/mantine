use-merged-ref

Edit this page
Source code
@mantine/hooks

Use multiple refs for one dom node

import { useMergedRef } from '@mantine/hooks';

Definition

Hook accepts any amount of refs and returns function that should be passed to dom node:

useMergedRef<T = any>(
  ...refs: React.ForwardedRef<T>[]
): (node: T) => void;

Usage

Use hook when you need to use more than one ref on single dom node, for example, when you want to use use-click-outside and use-focus-trap hooks and also get a ref for yourself:

import React, { useRef } from 'react';
import { useClickOutside, useMergedRef } from '@mantine/hooks';
import useFocusTrap from '@charlietango/use-focus-trap';


function Example() {
  const myRef = useRef();
  const useClickOutsideRef = useClickOutside(() => {});
  const focusTrapRef = useFocusTrap();
  const mergedRef = useMergedRef(myRef, useClickOutsideRef, focusTrapRef);


  return <div ref={mergedRef} />;
}
