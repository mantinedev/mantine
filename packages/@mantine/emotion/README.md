# @mantine/emotion

[![npm](https://img.shields.io/npm/dm/@mantine/emotion)](https://www.npmjs.com/package/@mantine/emotion)

Emotion bindings for @mantine/core

[View documentation](https://mantine.dev/)

## Installation

```bash
# With yarn
yarn add @mantine/emotion @mantine/core @mantine/hooks @emotion/react

# With npm
npm install @mantine/emotion @mantine/core @mantine/hooks @emotion/react
```
