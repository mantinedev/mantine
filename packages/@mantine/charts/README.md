# @mantine/charts

[![npm](https://img.shields.io/npm/dm/@mantine/charts)](https://www.npmjs.com/package/@mantine/charts)

Charts components built with recharts and Mantine

[View documentation](https://mantine.dev/)

## Installation

```bash
# With yarn
yarn add @mantine/charts @mantine/core @mantine/hooks recharts

# With npm
npm install @mantine/charts @mantine/core @mantine/hooks recharts
```
