# Can I remove MultiSelect placeholder when the component has selected values?
Learn why MultiSelect placeholder is not removed when values are selected and how to remove it with CSS

## Why is the MultiSelect placeholder not removed when values are selected?

The [MultiSelect](https://mantine.dev/core/multi-select) component uses the placeholder to indicate that
there are values available for selection. It is different from the [Select](https://mantine.dev/core/select)
component where the placeholder is removed when a value is selected – the user can select only one value.
