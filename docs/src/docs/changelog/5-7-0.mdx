## Responsive Header and Footer height

[Header and Footer](https://mantine.dev/core/app-shell/) components now support responsive height:

```tsx
import { Header } from '@mantine/core';

function Demo() {
  return <Header height={{ base: 50, sm: 60, lg: 70 }} />;
}
```

## Other changes

- [Collapse](https://mantine.dev/core/collapse/) component now supports `axis` prop, it is now possible to animate width
- [Button](https://mantine.dev/core/button/) component now supports `loaderPosition="center"`
- [Carousel](https://mantine.dev/others/carousel/) now supports `onSlideChange` prop
- [MantineProvider](https://mantine.dev/theming/mantine-provider/) now includes [theme.primaryColor](https://mantine.dev/theming/colors/#primary-color) validation â€“ it will throw an error if primary color was set to an invalid value
- [use-form](https://mantine.dev/form/use-form/) `onSubmit` can now be called without form event
- [Carousel](https://mantine.dev/others/carousel/) now supports `withKeyboardEvents` prop that allows to disable keyboard events handling
