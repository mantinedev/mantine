## Other changes

- [Grid.Col](https://mantine.dev/core/grid/) component no longer requires to set `span`, it is set to Grid `columns` by default
- [Dropzone](https://mantine.dev/others/dropzone/) component now supports `onReject` callback which is called when user tries to drop files that do not meet size or mime type requirements
- [Stepper](https://mantine.dev/core/stepper/) component now supports `allowStepSelect` prop on `Stepper.Step` component which can be used to prevent user from reaching next steps before completing previous
- [Accordion](https://mantine.dev/core/accordion/) component now supports changing headings level with `order` prop
- [Tabs](https://mantine.dev/core/tabs/) component now supports getting tab keys in `onTabChange` handler

## Bug fixes

- Fix incorrect default `shouldCreate` logic in creatable [Select](https://mantine.dev/core/select/)
- Fix incorrect border-radius in [TransferList](https://mantine.dev/core/transfer-list/) component
- Add color style to [UnstyledButton](https://mantine.dev/core/button/#unstyled-button) in dark theme
- Fix incorrect [Menu](https://mantine.dev/core/menu/) body background color in dark theme
