### New hooks and components

New [use-hotkeys](/hooks/use-hotkeys/) hook allows you to subscribe to multiple
hotkeys with single hook (document element) or function (any other element):

<Demo data={HooksDemos.useHotkeysDemo} demoProps={{ toggle: true }} />

New [use-intersection](/hooks/use-intersection/) hook allows you to get information
about intersection of given element with its scroll container or `document.body`:

<Demo data={HooksDemos.useIntersectionDemo} demoProps={{ toggle: true }} />

Get and set hash in url with new [use-hash](/hooks/use-hash/) hook:

<Demo data={HooksDemos.useHashDemo} demoProps={{ toggle: true }} />

### Other new features

- [Highlight](/core/highlight/) component now supports multiple substrings
  highlight and highlighting of the same string multiple times
- [Tabs](/core/tabs/) component now supports vertical orientation
- [Input](/core/input/) and all other components that use it (Select, Textarea,
  TextInput, Autocomplete, etc.) now supports headless variant without Mantine styles to help you create
  your own custom input styles with Styles API
