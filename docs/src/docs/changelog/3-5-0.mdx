## Other changes

- New [use-element-size](https://mantine.dev/hooks/use-element-size/) hook â€“ subscribe to element width and height with ResizeObserver, simpler alternative to [use-resize-observer](https://mantine.dev/hooks/use-resize-observer/)
- [Select](https://mantine.dev/core/select/) component now has an option to enable active item deselection when user clicks on active item
- [Skeleton](https://mantine.dev/core/skeleton/) component now supports animations disabling
- [Alert](https://mantine.dev/core/alert/), [Tooltip](https://mantine.dev/core/tooltip) and [Notification](https://mantine.dev/core/notification/) components now support `radius` prop
- [Alert](https://mantine.dev/core/alert) component now supports 3 variants: filled, light and outline
- [Center](https://mantine.dev/core/center/) component is now polymorphic, it supports `component` prop
- [Grid](https://mantine.dev/core/grid/) component now exposes `Col` component as `Grid.Col`
- [Progress](https://mantine.dev/core/progress/) and [RingProgress](https://mantine.dev/core/ring-progress/) components now support setting colors with css color values instead of allowing only values from `theme.colors`
- [Modal](https://mantine.dev/core/modal/) component now has an option to disable closing when escape key is pressed with `closeOnEscape={false}`
- `@mantine/core` package now exports `MantineProviderProps` type
- `@mantine/hooks` package now exports `UseForm` type

## 3.4.0 - 3.5.0 bug fixes

- Fix incorrect click outside events handling in [Modal](https://mantine.dev/core/modal/) and [Drawer](https://mantine.dev/core/drawer/) components
- Fix incorrect max-width calculated for [Group](https://mantine.dev/core/group/) with falsy children
- Fix incorrect animations applied to [Collapse](https://mantine.dev/core/collapse/) component with `animateOpacity` prop set to false
- Fix [HueSlider and AlphaSlider](https://mantine.dev/core/color-picker/) components breaking when used without `size` prop
- Fix `size` prop not applied to [Divider](https://mantine.dev/core/divider/) with label
- Fix `onChange` function called with incorrect precision in [NumberInput](https://mantine.dev/core/number-input/) component
- Fix Linux detected as undetermined is [use-os](https://mantine.dev/hooks/use-os/) hook
- Fix incorrect [MediaQuery](https://mantine.dev/core/media-query/) component `largerThan` and `smallerThan` props handling (**breaking bug fix**)
- Fix incorrect [DateRangePicker](https://mantine.dev/dates/date-picker/) types for strict TypeScript mode
- Fix unexpected vertical scrollbars in [TransferList](https://mantine.dev/core/transfer-list/) component
- Fix unexpected onChange function called onBlur in [DatePicker](https://mantine.dev/dates/date-picker/) component
