## validateInputOnBlur

[use-form](https://mantine.dev/form/use-form/) now supports `validateInputOnBlur` [option](https://mantine.dev/form/validation/#validate-fields-on-blur),
it works similar to `validateInputOnChange`:

<Demo data={FormDemos.blurFieldValidation} />

## Non-linear Slider scale

[Slider and RangeSlider](https://mantine.dev/core/slider/) components now support non-linear scale:

<Demo data={SliderDemos.scale} />

## Switch.Group component

New [Switch.Group](https://mantine.dev/core/switch/) component lets you organize `Switch` components the same way as `Checkbox.Group` and `Radio.Group`:

<Demo data={SwitchDemos.groupConfigurator} />

## Controlled Select/MultiSelect search value

[Select](https://mantine.dev/core/select/) and [MultiSelect](https://mantine.dev/core/multi-select/) search value can now be controlled:

<Demo data={SelectDemos.searchableControlled} />

## Controlled PasswordInput visibility

[PasswordInput](https://mantine.dev/core/password-input/) now supports controlled visibility state with `visible` and `onVisibilityChange` props,
for example, the props can be used to sync visibility state between two inputs:

<Demo data={PasswordInputDemos.controlledVisibility} />

## New Mantine UI components

10 new components were added to [Mantine UI](https://ui.mantine.dev/), view changelog [here](https://ui.mantine.dev/changelog/september-2022)

## Other changes

- It is now possible to change [static classes prefix](https://mantine.dev/styles/styles-api/#static-class-names) with [emotion cache key](https://mantine.dev/theming/emotion-cache/)
- [use-local-storage](https://mantine.dev/hooks/use-local-storage/) hook now supports removing value from `localStorage`
- [Pagination](https://mantine.dev/core/pagination/) component now supports `disabled` prop
- [ColorPicker](https://mantine.dev/core/color-picker/) and [ColorInput](https://mantine.dev/core/color-input/) components now support `onChangeEnd` callback
- [Group](https://mantine.dev/core/group/) component now uses child selector instead of passing `className` to children
- [Dropzone](https://mantine.dev/others/dropzone/) component now supports `useFsAccessApi` prop
- [SimpleGrid](https://mantine.dev/core/simple-grid/) component now supports `verticalSpacing` prop
- [ThemeIcon](https://mantine.dev/core/theme-icon/) component now supports `variant="default"`
