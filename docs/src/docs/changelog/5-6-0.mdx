## Form index reference in validateInputOnBlur and validateInputOnChange

You can now use `FORM_INDEX` in [use-form](https://mantine.dev/form/validation/) to validate nested array fields with `validateInputOnBlur` and `validateInputOnChange` settings:

<Demo data={FormDemos.liveFieldValidation} />

## use-form transformValues

[use-form](https://mantine.dev/form/values/) now supports `transformValues` options, it transforms values before they get submitted in `onSubmit` handler.
For example, it can be used to merge several fields into one or to convert types:

<Demo data={FormDemos.transformValues} />

## Other changes

- [CloseButton](https://mantine.dev/core/close-button/) is now a polymorphic component
- [Autocomplete](https://mantine.dev/core/autocomplete/), [ColorInput](https://mantine.dev/core/color-input/), [FileInput](https://mantine.dev/core/file-input/), [JsonInput](https://mantine.dev/core/json-input/) components now support `readOnly`
- [FileButton](https://mantine.dev/core/file-button/) now supports `disabled` prop
- [MediaQuery](https://mantine.dev/core/media-query/) component `styles` prop can now use `theme`
- [use-timeout](https://mantine.dev/hooks/use-timeout/) hook now supports parameters in callback function
