## Usage

Hook detects user os. Possible values are: `undetermined`, `macos`, `ios`, `windows`, `android`, `linux`.
If os cannot be identified as well as during server side rendering `undetermined` will be returned.

<Demo data={HooksDemos.useOsDemo} />

## TypeScript

You can import `OS` union type from `@mantine/hooks`:

```tsx
import type { OS } from '@mantine/hooks';
// OS type is 'undetermined' | 'macos' | 'ios' | 'windows' | 'android' | 'linux'
```

## Definition

```tsx
function getOS(): 'undetermined' | 'macos' | 'ios' | 'windows' | 'android' | 'linux';
```
