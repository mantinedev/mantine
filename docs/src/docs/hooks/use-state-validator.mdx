## Usage

use-state-validator hook is a `useState` wrapper with validation for the state.

```tsx
const [value, setValue, isValid] = useStateValidator('initialValue', (val) => typeof val === 'string');

value // -> initialValue
isValid // -> true

setValue('nextValue') // -> nextValue
isValid // -> true

setValue(undefined); // -> undefined
isValid // => false
```

You can also pass a callback to `setValue`.

```tsx
setValue((prevValue) => prevValue + ' Updated');
```

## Definition

```tsx
function useStateValidator<T>(
  initialValue: T | null | undefined,
  validator: (val: T | null | undefined) => boolean
): readonly [T, ((T | null | undefined | ((prevValue: T) => T) => void), boolean];

```
