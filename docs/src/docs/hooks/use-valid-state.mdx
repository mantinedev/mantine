## Usage

use-valid-state works similar to `useState` but checks the value if it is valid based on the `rule` function.

```tsx
const [validValue, setValue, value] = useValidState((value) => value === 'valid', 'valid');

validValue; // -> valid
value; // -> valid

setValue('invalid');

validValue; // -> valid
value; // -> invalid
```

## Definition

```tsx
function useValidState<T>(
  validation: (value: T) => boolean,
  initialValue?: T
): readonly [validValue: T, setValue: (value: T) => void, value: T];
```
