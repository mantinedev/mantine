## Usage

This hook returns a boolean state value which keeps track of inactivity.

<Demo data={useIdleHook} />

## API

Hook accepts one argument:

- `timeout` â€“ period of inactivity or idle time.

Hook returns a boolean value depicting the idle status
- true - idle >= timeout.
- false - on any activity.

```tsx
const idleStatus = useIdle(1000);//timeout

//initial value or on being idle for >= 1 sec
idleStatus -> true

//On any activity
idleStatus -> false
```

### Definition

```tsx
function useIdle(
  timeout: number
): boolean;
```
