## API

```tsx
const { start, clear } = useTimeout(fn, delay, { autoInvoke: true });
```

Arguments:

- `fn` – function that will be called after the timer expires
- `interval` – amount of milliseconds the timer should wait before the specified function is executed.
- `options: { autoInvoke }` - If the timer should be invoked on component mount, defaults to true.   

Return object:

- `start` - to start the timer.
- `clear` – to cancel the timer.

## Definition

```tsx
function useInterval(
  fn: () => void,
  delay: number,
  options: {
    autoInvoke: boolean;
  }
): {
  start: () => void;
  clear: () => void;
};
```
