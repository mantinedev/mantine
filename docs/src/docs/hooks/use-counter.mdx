## Usage

use-counter helps manage a counter using a set of operations.

<Demo data={HooksDemos.useCounterDemo} />

## API

```tsx
const {
  count, // -> counter value
  {
    inc,
    dec,
    set,
    reset
  }, // -> operations applicable on counter
} = useCounter(initialValue, { min, max });
``` 

## Definition

```tsx
function useCounter(initialValue?: number, options?: { min?:number, max?:number }): [
  count:number, {
    inc: (delta?: number) => void,
    dec: (delta?: number) => void,
    set: (value: number) => void,
    reset: () => void
}];
```
