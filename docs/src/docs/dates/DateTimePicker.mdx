## Accessibility

### DatePicker labels

Follow [DatePicker](/dates/date-picker/#accessibility) documentation
to add all required labels to the year picker inside popover.

### TimeInput label

Use `timeInputProps` to set `aria-label` of `TimeInput`:

```tsx
<DatePickerInput timeInputProps={{ 'aria-label': 'Pick time' }} />
```

### Submit button label

Use `submitButtonProps` to add label to the submit button:

```tsx
<DatePickerInput submitButtonProps={{ 'aria-label': 'Submit' }} />
```

### Input label

Set `aria-label` in case you use component without `label`:

```tsx
<DatePickerInput /> // -> not ok, input is not labeled
<DatePickerInput label="My input" /> // -> ok, input and label is connected
<DatePickerInput aria-label="My input" /> // -> ok, label is not visible but will be announced by screen reader
```
