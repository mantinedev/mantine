---
title: 'Popover component error'
error: 'Popover component was not found in the tree'
---

## Error source

You have rendered `Popover.Target` or `Popover.Dropdown` component outside of [Popover](/core/popover/) component context:

```tsx
import { Popover } from '@mantine/core';

function DemoWithError() {
  return (
    <>
      <Popover.Target>
        <button>Target</button>
      </Popover.Target>

      <Popover.Dropdown>Dropdown</Popover.Dropdown>
    </>
  );
}
```

## How to fix

Make sure that `Popover.Target` and `Popover.Dropdown` components are rendered within [Popover](/core/popover/) component
â€“ all of them require Popover context to work correctly:

```tsx
import { Popover } from '@mantine/core';

function Demo() {
  return (
    <Popover>
      <Popover.Target>
        <button>Target</button>
      </Popover.Target>

      <Popover.Dropdown>Dropdown</Popover.Dropdown>
    </Popover>
  );
}
```
