---
title: 'Accordion component error'
error: 'Accordion.Item component was rendered with invalid value or without value'
---

## Error source

`value` is a required prop for `Accordion.Item` component.
`value` prop should be a string, other types are not supported:

```tsx
import { Accordion } from '@mantine/core';

function DemoWithError() {
  return (
    <Accordion>
      <Accordion.Item>Without value</Accordion.Item>
      <Accordion.Item value={null}>Invalid value</Accordion.Item>
      <Accordion.Item value={2}>Invalid value</Accordion.Item>
    </Accordion>
  );
}
```

## How to fix

Add string `value` to all `Accordion.Item` components:

```tsx
import { Accordion } from '@mantine/core';

function Demo() {
  return (
    <Accordion>
      <Accordion.Item value="item-1">Valid value</Accordion.Item>
      <Accordion.Item value="item-2">Valid value</Accordion.Item>
      <Accordion.Item value="item-3">Valid value</Accordion.Item>
    </Accordion>
  );
}
```
