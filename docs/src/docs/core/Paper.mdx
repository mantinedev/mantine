## Usage

Paper component renders white (or `theme.colors.dark[7]` for dark theme)
background with shadow, border-radius and padding from theme.

<Demo data={PaperDemos.configurator} />

## Shadow

You can use predefined shadows defined in [theme](/theming/extend-theme/#spacing-radius-and-shadows) or your own:

```tsx
<Paper shadow="xs">
  Paper with predefined shadow from theme
</Paper>

<Paper shadow="13px 18px 25px #e5e5e5, 1px 3px 3px #e5e5e5, -1px 3px 3px #e5e5e5">
  Paper with custom shadow
</Paper>
```

## Padding

Paper has predefined padding: xs, sm, md, lg, xl defined in [theme.spacing](/theming/extend-theme/#spacing-radius-and-shadows).
Alternatively, you can use a number to set padding in px:

```tsx
<Paper padding="sm" /> // -> theme predefined small spacing
<Paper padding={20} /> // -> { padding: '20px' }
```

## Radius

xs, sm, md, lg, xl radius values are defined in [theme.radius](/theming/extend-theme/#spacing-radius-and-shadows).
Alternatively, you can use a number to set radius in px:

```tsx
<Paper radius="lg" /> // -> theme predefined large radius
<Paper radius={0} /> // -> { borderRadius: 0 }
```

## Get root element ref

You can get root element ref with `ref` prop:

```tsx
import { useRef } from 'react';
import { Paper } from '@mantine/core';

function Demo() {
  const ref = useRef();
  return <Paper ref={ref} />;
}
```

## TypeScript

### Usage with custom component

You need to pass additional type to use custom component with TypeScript
in case you need full type coverage for events and `ref`:

```tsx
// With element
<Paper<'a'>
  component="a"
  onClick={(event) => console.log(event)}
  ref={(node) => {
    window.myRef = node;
  }}
/>

// With component
<Paper<typeof Link>
  component={Link}
  onClick={(event) => console.log(event)}
  ref={(node) => {
    window.myRef = node;
  }}
/>
```

### Component props type

Import component props type:

```tsx
import type { PaperProps } from '@mantine/core';
```

`PaperProps` is a generic type, it accepts either a string or component type as single type argument:

```tsx
// PaperProps with html tag, will extend React.ComponentPropsWithoutRef<'a'>
type AnchorPaperProps = PaperProps<'a'>;

// PaperProps with custom component, will extend React.ComponentPropsWithoutRef<typeof MyOtherComponent>
type CustomPaperProps = PaperProps<typeof MyOtherComponent>;
```

See [TypeScript guide](/pages/basics/#typescript) to learn how to utilize props type with custom component.
