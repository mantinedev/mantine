## Usage

Paper component renders white (or `theme.colors.dark[7]` for dark theme)
background with shadow, border-radius and padding from theme.

<Demo data={PaperDemos.configurator} />

## Polymorphic component

Paper is a [polymorphic component](/guides/polymorphic/), you can change root element:

```tsx
import { Paper } from '@mantine/core';

function Demo() {
  return (
    <Paper component="a" href="https://mantine.dev/core/">
      Anchor element
    </Paper>
  );
}
```

## Get ref

```tsx
import { useRef } from 'react';
import { Paper } from '@mantine/core';

function Demo() {
  const ref = useRef<HTMLDivElement>(null);
  return <Paper ref={ref} />;
}
```
