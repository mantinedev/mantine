## Usage

Use Checkbox to capture boolean value input from user.
For better accessibility set `label` prop, it will add associated label element.

## Sizes

Checkbox has 5 predefined sizes: xs, sm, md, lg, xl.
Size defines label font-size, input width and height:

```tsx
<Checkbox size="xl" /> // -> predefined xl size
```

<Demo data={CheckboxDemos.sizes} />

You can get checkbox sizes by importing `CHECKBOX_SIZES`:

```tsx
import { CHECKBOX_SIZES } from '@mantine/core';
```

## Indeterminate state

Checkbox supports indeterminate state. When `indeterminate` prop is set to true,
`checked` prop is ignored:

<Demo data={CheckboxDemos.indeterminate} />

## Controlled

```tsx
import { useState } from 'react';
import { Checkbox } from '@mantine/core';

function Demo() {
  const [checked, setChecked] = useState(false);
  return (
    <Checkbox checked={checked} onChange={(event) => setChecked(event.currentTarget.checked)} />
  );
}
```

## Get input ref

You can get input ref by setting `ref` prop:

```tsx
import { useRef } from 'react';
import { Checkbox } from '@mantine/core';

function Demo() {
  const ref = useRef();
  return <Checkbox ref={ref} />;
}
```

## Accessibility

Provide `aria-label` in case you use checkbox without label for screen reader support:

```tsx
<Checkbox label="My checkbox" />; // -> ok, input and label is connected
<Checkbox />; // not ok, input is not labeled
<Checkbox aria-label="My checkbox" />; // -> ok, label is not visible but will be announced by screen reader
```

## TypeScript

### Component props

Import component props type:

```tsx
import type { CheckboxProps } from '@mantine/core';
```

`CheckboxProps` extends `React.ComponentPropsWithoutRef<'input'>`, all other props are available under _Component props_ tab.

### CheckboxStylesNames type

`CheckboxStylesNames` type is a union with Checkbox [Styles API](/theming/styles-api/) names:

```tsx
import type { CheckboxStylesNames } from '@mantine/core';

const checkboxClasses: Partial<Record<CheckboxStylesNames, 'string'>> = {
  icon: 'checkbox-icon',
  label: 'checkbox-label',
};

<Checkbox classNames={checkboxClasses} />;
```

See full styles descriptions list under _Styles API_ tab.
