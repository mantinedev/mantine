## Usage

CloseButton is a premade [ActionIcon](/core/action-icon/) with close icon.
Component accepts the same props as [ActionIcon](/core/action-icon/)
with additional `iconSize` prop to control icon width and height:

<Demo data={CloseButtonDemos.usage} />

## Accessibility

CloseButton does not have an associated label, set `title` or `aria-label` props to make it visible for screen readers:

```tsx
<CloseButton /> // -> not visible to screen reader
<CloseButton title="Settings" /> // -> ok
<CloseButton aria-label="Settings" /> // -> ok
```
