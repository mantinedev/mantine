## Installation

Package depends on [react](https://www.npmjs.com/package/react), [react-dom](https://www.npmjs.com/package/react-dom) and [@mantine/hooks](https://www.npmjs.com/package/@mantine/hooks).

Install with npm:

```sh
npm install @mantine/core @mantine/hooks
```

Install with yarn:

```sh
yarn add @mantine/core @mantine/hooks
```

## Browser support

All components are tested to work in all major modern browsers.
Some components depend on wep APIs that may not be supported by old browsers (e.g. [window.matchMedia](https://developer.mozilla.org/en-US/docs/Web/API/Window/matchMedia) and [ResizeObserver](https://developer.mozilla.org/en-US/docs/Web/API/ResizeObserver)),
if you encounter any issues with browsers compatibility [open an issue](https://github.com/mantinedev/mantine/issues/new).
