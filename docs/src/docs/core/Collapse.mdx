## Change transition

Set following props to control transition:

- `transitionDuration` – duration in ms
- `transitionTimingFunction` – timing function (ease, linear, etc.), defaults to `ease`
- `onTransitionEnd` – called when transition ends (both open and close)

## TypeScript

Import component props type:

```tsx
import type { CollapseProps } from '@mantine/core';
```

`CollapseProps` extends `React.ComponentPropsWithoutRef<'div'>`, all other props are available under _Component props_ tab.
