## Usage

You can use any React node as icon by passing it as a child to ActionIcon component.
Note that component **does not control icon size** and you need to specify it manually on icon component for better fit:

```tsx
<ActionIcon>
  <YourIcon style={{ width: 16, height: 16 }} />
</ActionIcon>
```

You can use icons from any react icons library, for example:

- [radix icons](https://icons.modulz.app/) (used in mantine packages)
- [feather](https://feathericons.com/)
- [react-icons](https://react-icons.github.io/react-icons)

## Size

Note that component **does not control icon size** and you need to specify it manually on icon component for better fit.
ActionIcon has predefined sizes: xs, sm, md, lg, xl.
You can pass size as number and it will be set as width and height in px:

```tsx
<ActionIcon size="sm" /> // -> predefined small size
<ActionIcon size={50} /> // -> { width: 50, height: 50 }
```

You can get default sizes values by importing `ACTION_ICON_SIZES`:

## Radius

Radius has predefined values: xs, sm, md, lg, xl which are defined in <GatsbyLink to="/pages/theming/">theme.radius</GatsbyLink>.
Alternatively you can use number to set border-radius in px:

```tsx
<ActionIcon radius="lg" /> // -> theme predefined large radius
<ActionIcon radius={10} /> // -> { borderRadius: 10 }
<ActionIcon radius={0} /> // -> remove border-radius
```

## Get element ref

You can get button ref with `elementRef` prop:

```tsx
import React, { useRef } from 'react';
import { ActionIcon } from '@mantine/core';

function Demo() {
  const ref = useRef();
  return <ActionIcon elementRef={ref} />;
}
```

## Accessibility

ActionIcon renders a regular button element.
Make sure to include `title` or `aria-label` props for screen reader support
as by design element does not have text content.

```tsx
// Title will show message on hover and announce control with screen reader
<ActionIcon title="Settings">
  <GearIcon />
</ActionIcon>

// aria-label will only announce control with screen reader
<ActionIcon aria-label="Settings">
  <GearIcon />
</ActionIcon>
```
