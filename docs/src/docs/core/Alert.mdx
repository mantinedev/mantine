## Usage

Use Alert to attract user attention with important message.
Alert is designed to be static, if you need more dynamic variant use [Notification](/core/notification/) component.

## Color

You can use any color defined in [theme.colors](/theming/extend-theme/#default-colors).
By default Alert will have `theme.primaryColor` color:

## Accessibility

- Root element role set to `alert`
- Set `closeButtonLabel` prop to make close button visible for screen readers
