## Size

Burger has predefined sizes: xs, sm, md, lg, xl.
You can pass size as number and it will be set as width and height in px.

```tsx
<Burger size="sm" /> // -> predefined small size
<Burger size={50} /> // -> { width: 50, height: 50 }
```

## Get element ref

You can get button ref by setting `elementRef` prop:

```tsx
import React, { useRef } from 'react';
import { Burger } from '@mantine/core';

function Demo() {
  const ref = useRef();
  return <Burger elementRef={ref} opened />;
}
```

## Accessibility

Burger renders a regular button element.
Make sure to include `title` or `aria-label` props for screen reader support
as by design element does not have text content.

```tsx
// Title will show message on hover and announce control with screen reader
<Burger title="Open navigation" />

// aria-label will only announce control with screen reader
<Burger aria-label="Open navigation" />
```
