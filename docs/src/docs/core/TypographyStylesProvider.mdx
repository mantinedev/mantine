## Usage

Mantine does not expose global styles.
Use TypographyStylesProvider to add Mantine typography styles to your html content:

```tsx
import { TypographyStylesProvider } from '@mantine/core';

function Demo() {
  return (
    <TypographyStylesProvider>
      <div dangerouslySetInnerHTML={{ __html: '<p>Your html here</p>' }} />
    </TypographyStylesProvider>
  );
}
```

- lists
- blockquotes
- tables
- links
- images

- code and pre
