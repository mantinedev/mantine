## Size

Size controls loader height, width is calculated dynamically. Loader has predefined sizes: xs, sm, md, lg, xl.
Alternatively, you can use a number to set height in px:

```tsx
<Loader size="sm" /> // -> predefined small size
<Loader size={50} /> // -> { height: 50 }
```

You can get predefined sizes by importing `LOADER_SIZES`:

```tsx
import { LOADER_SIZES } from '@mantine/core';
```
