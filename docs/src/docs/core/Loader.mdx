## Usage

By default, loader will be rendered with `theme.primaryColor`. You can choose any color defined in `theme.colors`:

<Demo data={LoaderDemos.configurator} />

## Configure default loader

You can configure default loader in [MantineProvider](/theming/mantine-provider/),
it will be used in Loader and [LoadingOverlay](/core/loading-overlay/) components by default:

```tsx
import { MantineProvider } from '@mantine/core';

function Demo() {
  return (
    <MantineProvider theme={{ loader: 'bars' }}>
      <App />
    </MantineProvider>
  );
}
```

## Size

Size controls loader height or width (depends on loader variant). Loader has predefined sizes: xs, sm, md, lg, xl.
Alternatively, you can use a number to set width and height:

```tsx
<Loader size="sm" /> // -> predefined small size
<Loader size={48} /> // -> size in rem -> 48 -> 3rem
```
