## Usage

Anchor is a wrapper around [Text](/core/text/) component with `component` prop set to `a` by default:

<Demo data={AnchorDemos.usage} />

## Usage with react-router and other libraries

You can use Anchor with react-router Link component and other similar components by setting `component` prop:

```tsx
import { Link } from 'react-router-dom';
import { Anchor } from '@mantine/core';

function Demo() {
  return (
    <Anchor component={Link} to="/react-router">
      React router link
    </Anchor>
  );
}
```

## Typescript

Import component props type:

```tsx
import type { AnchorProps } from '@mantine/core';
```

`AnchorProps` is a generic type, it accepts either a string or component type as single type argument:

```tsx
// AnchorProps with html tag, will extend React.ComponentPropsWithoutRef<'a'>
type MyAnchorProps = AnchorProps<'a'>;

// AnchorProps with custom component, will extend React.ComponentPropsWithoutRef<typeof Link>
type CustomAnchorProps = AnchorProps<typeof Link>;
```

See [TypeScript guide](/pages/basics/#typescript) to learn how to utilize props type with custom component.
