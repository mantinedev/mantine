## Usage

Anchor is a wrapper around [Text](/core/text/) component with `component` prop set to `a` by default.
It supports the same props as Text component.

<Demo data={AnchorDemos.usage} />

## Polymorphic component

Anchor is a [polymorphic component](/guides/polymorphic/), you can change root element:

```tsx
import { Anchor } from '@mantine/core';

function Demo() {
  return (
    <Anchor component="button" type="button">
      Anchor as button
    </Anchor>
  );
}
```
