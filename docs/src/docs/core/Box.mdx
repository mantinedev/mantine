## Usage

Box allows you to use [sx prop](/styles/sx) with any element or component.
Box itself does not include any styles.

<Demo data={BoxDemos.usage} />

## Custom component

To use Box with custom component or element set `component` prop to tag name or react component:

```tsx
import { ActionIcon } from '@mantine/core';
import { Link } from 'react-router-dom';

function Demo() {
  return (
    <>
      <ActionIcon<'a'> component="a" href="#" />
      <ActionIcon<typeof Link> component={Link} to="/hello" />
    </>
  );
}
```
