## Usage

Box allows you to use [sx prop](/styles/sx) with any element or component.
Box itself does not include any styles.

<Demo data={BoxDemos.usage} />

## Polymorphic component

Box is a [polymorphic component](/guides/polymorphic/), you can change root element:

```tsx
import { Box } from '@mantine/core';
import { Link } from 'react-router-dom';

function Demo() {
  return (
    <>
      <Box component="a" href="#">
        Box as anchor
      </Box>
      <Box component={Link} to="/hello">
        Box as react-router-dom link
      </Box>
    </>
  );
}
```
