## TypeScript

### Component props type

Import component props type:

```tsx
import type { BreadcrumbsProps } from '@mantine/core';
```

`BreadcrumbsProps` extends `React.ComponentPropsWithoutRef<'div'>`, all other props are available under _Component props_ tab.

### BreadcrumbsStylesNames type

`BreadcrumbsStylesNames` type is a union with Breadcrumbs [Styles API](/theming/styles-api/) names:

```tsx
import type { BreadcrumbsStylesNames } from '@mantine/core';

const breadcrumbsClasses: Partial<Record<BreadcrumbsStylesNames, 'string'>> = {
  breadcrumb: 'breadcrumbs-breadcrumb',
  separator: 'breadcrumbs-separator',
};

<Breadcrumbs classNames={breadcrumbsClasses} />;
```

See full styles descriptions list under _Styles API_ tab.
