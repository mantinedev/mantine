## Usage

Use PasswordInput when you need to capture password from user.
Component provides option to toggle password visibility, if you do not this feature, use [TextInput](/core/text-input/) component with `type="password"`.
Component supports all props from [Input](/core/input/) and [InputWrapper](/core/input-wrapper/) components.

## Controlled

```tsx
import { useState } from 'react';
import { PasswordInput } from '@mantine/core';

function Demo() {
  const [value, setValue] = useState('');
  return <PasswordInput value={value} onChange={(event) => setValue(event.currentTarget.value)} />;
}
```

## Get input ref

You can get input ref by passing `ref` prop to PasswordInput component:

```tsx
import { useRef } from 'react';
import { PasswordInput } from '@mantine/core';

function PasswordInputDemo() {
  const ref = useRef(null);
  return <PasswordInput ref={ref} />;
}
```

## Accessibility

Provide `aria-label` in case you use component without label for screen reader support:

```tsx
<PasswordInput label="Your password" />; // -> ok, input and label is connected
<PasswordInput />; // not ok, input is not labeled
<PasswordInput aria-label="Your password" />; // -> ok, label is not visible but will be announced by screen reader
```

## TypeScript

Import component props type:

```tsx
import type { PasswordInputProps } from '@mantine/core';
```

`PasswordInputProps` extends `React.ComponentPropsWithoutRef<'input'>`, all other props are available under _Component props_ tab.
