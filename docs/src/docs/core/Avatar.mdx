## Radius

xs, sm, md, lg, xl radius values are defined in <GatsbyLink to="/pages/theming/">theme.radius</GatsbyLink>.
Alternatively you can use number to set radius in px:

```tsx
<Avatar radius="lg" /> // -> theme predefined large radius
<Avatar radius={1000} /> // -> { borderRadius: 1000 }
```

## Size

Avatar has predefined sizes: xs, sm, md, lg, xl.
You can pass size as number and it will be set as width and height in px.

```tsx
<Avatar size="sm" /> // -> predefined small size
<Avatar size={50} /> // -> { width: 50, height: 50 }
```

You can get default sizes for Avatar by importing `AVATAR_SIZES`:

```tsx
import { AVATAR_SIZES } from '@mantine/core';
```

## Placeholder

If `src` prop is not set, equals to null or image cannot be loaded, placeholder icon will be rendered instead.

## Custom placeholders

You can rendered any React node instead of placeholder icon by passing it as `children`. Usually icon or initials are used in this case.

You can choose any color defined in <GatsbyLink to="/pages/theming/">theme</GatsbyLink> for Avatar placeholder.

## Accessibility

Avatar renders img html element. It is important to add alt text.
In case of image load failing alt will be used as `title` for placeholder.
