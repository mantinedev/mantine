## Usage

Use ThemeIcon to display an svg icon with theme color background and color. You can use icons from any react icons library, for example:

- [radix icons](https://icons.modulz.app/) (used in mantine packages)
- [feather](https://feathericons.com/)
- [react-icons](https://react-icons.github.io/react-icons)

## Color

You can use any color from <GatsbyLink to="/theming/extend-theme/#default-colors">theme.colors</GatsbyLink>,

## Size

Note that component **does not control icon size** and you need to specify it manually on icon component for better fit.
ThemeIcon has 5 predefined sizes: xs, sm, md, lg, xl. Size defines background width and height.
Alternatively, you can use a number to set width and height in px:

```tsx
<ThemeIcon size="lg" /> // -> theme predefined large size
<ThemeIcon size={10} /> // -> { width: 10, height: 10 }
```

You can get default sizes for action icon by importing `THEME_ICON_SIZES`:


## Radius

xs, sm, md, lg, xl radius values are defined in <GatsbyLink to="/theming/extend-theme/#spacing-radius-and-shadows">theme.radius</GatsbyLink>.
Alternatively, you can use a number to set radius in px:

```tsx
<ThemeIcon radius="lg" /> // -> theme predefined large radius
<ThemeIcon radius={10} /> // -> { borderRadius: 10 }
<ThemeIcon radius={0} /> // -> remove border-radius
```
