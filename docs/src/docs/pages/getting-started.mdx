## Get started without framework

<PackagesInstallation />

Wrap your application root component with [MantineProvider](/theming/mantine-provider/):

```tsx
import { MantineProvider } from '@mantine/core';

function Demo() {
  return (
    <MantineProvider withGlobalStyles withNormalizeCSS>
      <App />
    </MantineProvider>
  );
}
```

All set! You can now import and use Mantine components anywhere in your application:

```tsx
import { Button } from '@mantine/core';

function Demo() {
  return <Button>Click me!</Button>;
}
```
