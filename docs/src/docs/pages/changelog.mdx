### API Changes

- Prop typo fix in <GatsbyLink to="/core/checkbox/">Checkbox</GatsbyLink> component (~~intermediate~~
  -> indeterminate)
- Hr component replaced with <GatsbyLink to="/core/divider/">Divider</GatsbyLink> to support more options
- ElementsGroup component replaced with <GatsbyLink to="/core/group/">Group</GatsbyLink> to support all elements positioning
- Custom component is now supported by <GatsbyLink to="/core/action-icon/">ActionIcon</GatsbyLink> and <GatsbyLink to="/core/menu">MenuItem</GatsbyLink> components
- <GatsbyLink to="/core/modal/">Modal</GatsbyLink> and <GatsbyLink to="/core/drawer/">
    Drawer
  </GatsbyLink> components are wrapped with <GatsbyLink to="/core/portal/">Portal</GatsbyLink> by default
- <GatsbyLink to="/core/loading-overlay">LoadingOverlay</GatsbyLink> now supports custom loaders

## Other changes

- Significant performance improvements with react-jss useStyles props memoization
- Dark theme improvements, all components now use more balanced transparent colors in dark theme
