### Other changes

- <GatsbyLink to="/core/menu/">Menu</GatsbyLink> and <GatsbyLink to="/core/radio-group/">
    RadioGroup
  </GatsbyLink> components now works differently with Styles API – all styles and classNames should be
  added to root Menu/RadioGroup component instead of MenuItem/Radio
- <GatsbyLink to="/core/card/">Card</GatsbyLink> component no longer has context styles for images and
  dividers, use new CardSection component instead
- @mantine/tag-picker package is deprecated – use <GatsbyLink to="/core/multi-select/">MultiSelect</GatsbyLink> instead
- You can now customize shadow and padding in <GatsbyLink to="/core/modal/">Modal</GatsbyLink> component
- <GatsbyLink to="/core/button/">Button</GatsbyLink> now supports compact option which reduces its height
  and horizontal padding
- <GatsbyLink to="/core/image/">Image</GatsbyLink> now supports adding figcaption
- New <GatsbyLink to="/core/mark/">Mark</GatsbyLink> component – mark tag with theme styles
- <GatsbyLink to="/core/menu/">Menu</GatsbyLink> component now supports labels in menu body and behavior
  customization (focus trap and close on scroll)
- 4 new premade pop-\* transitions were added to <GatsbyLink to="/core/transition/">Transition</GatsbyLink> component
- New <GatsbyLink to="/hooks/use-logger/">use-logger</GatsbyLink> hook will log given values each time component renders
- <GatsbyLink to="/core/select/">Select</GatsbyLink>, <GatsbyLink to="/core/native-select/">
    NativeSelect
  </GatsbyLink> and <GatsbyLink to="/core/autocomplete/">Autocomplete</GatsbyLink> components now
  support data as an array of strings
- New <GatsbyLink to="/hooks/use-shallow-effect/">use-shallow-effect</GatsbyLink> hook – a drop in replacement for useEffect with shallow comparison instead of referential
- <GatsbyLink to="/hooks/use-scroll-lock/">use-scroll-lock</GatsbyLink> hook now support state managing
  inside hook

## 2.0.0 – July 2021

### Breaking changes

- All inputs no longer support `inputStyle` and `inputClassName` props, this feature was replaced by <GatsbyLink to="/theming/styles-api/">Styles API</GatsbyLink>
- <GatsbyLink to="/core/burger/">Burger</GatsbyLink> component no longer uses colors from theme, use
  hex color value instead
- Select component was renamed to <GatsbyLink to="/core/native-select">NativeSelect</GatsbyLink>
- `ElementsGroup`, `Hr` and `CardsGrid` components are replaced and no longer exported from @mantine/core package
- All inputs variant is set by default to filled in dark theme

### New theming options

- @mantine/core package now exports <GatsbyLink to="/theming/global-styles/">GlobalStyles and NormalizeCSS</GatsbyLink> components to include global styles
- New <GatsbyLink to="/theming/styles-api/">Styles API</GatsbyLink> allows you to style any part of Mantine components with inline styles or classes

<Demo data={TabsDemos.stylesApi} />

### @mantine/core changes

- <GatsbyLink to="/core/tabs/">Tabs</GatsbyLink> component now supports unstyled and pills variants
- All inputs now support `size` prop
- <GatsbyLink to="/core/drawer/">Drawer</GatsbyLink> component now support title and close button

### New @mantine/hooks hooks

- <GatsbyLink to="/hooks/use-did-update">use-did-update</GatsbyLink> – useEffect replacement when you
  do not want hook to be called on mount
- <GatsbyLink to="/hooks/use-isomorphic-effect">use-isomorphic-effect</GatsbyLink> – switch between useEffect
  during server side rendering and useLayoutEffect after hydration
- <GatsbyLink to="/hooks/use-force-update">use-force-update</GatsbyLink> – force component to rerender
- <GatsbyLink to="/hooks/use-toggle">use-toggle</GatsbyLink> – switch state between two values
- <GatsbyLink to="/hooks/use-uncontrolled">use-uncontrolled</GatsbyLink> – manage state for both controlled
  and uncontrolled components
- <GatsbyLink to="/hooks/use-window-scroll">use-window-scroll</GatsbyLink> – get current scroll position
  and handler to scroll to given position
