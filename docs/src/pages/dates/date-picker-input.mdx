## DatePicker props

`DatePickerInput` supports most of the [DatePicker](/dates/date-picker/) props,
read through [DatePicker](/dates/date-picker/) documentation to learn about all component features that are not listed on this page.

## Open picker in modal

By default, [DatePicker](/dates/date-picker/) is rendered inside [Popover](/core/popover/).
You can change that to [Modal](/core/modal/) by setting `dropdownType="modal"`:

## Value formatter

`valueFormatter` is a more powerful alternative to `valueFormat` prop.
It allows formatting value label with a custom function.
The function is the same for all component types (`default`, `multiple` and `range`)
â€“ you need to perform additional checks inside the function to handle different types.
